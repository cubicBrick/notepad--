Notepad-- (notepad minus minus): Like notepad++, but worse
| Notepad++ feature | Notepad-- feature |
|-------------------|-------------------|
|Open source| Closed source|
|Very functional|Not functional|
|Not a game|A game|
|You can use a keyboard|No keyboards allowed|
|Does not have attack shark|Has attack shark|
|Syntax highlighting|No highlighting|
|Does not drive you crazy|Drives you crazy|

Notepad-- supports python 3.12-3.14 and windows, linx, and macOS.
