# The Notepad-- Game (v2.1.1)

Notepad-- (notepad minus minus): Like notepad++, but worse
| Notepad++ feature | Notepad-- feature |
|-------------------|-------------------|
|Open source| Closed source|
|Very functional|Not functional|
|Not a game|A game|
|You can use a keyboard|No keyboards allowed|
|Does not have attack shark|Has attack shark|
|Syntax highlighting|No highlighting|
|Does not drive you crazy|Drives you crazy|

Notepad-- supports python 3.12-3.14 and windows, linx, and macOS.

# How to play Notepad-- (EXECUTABLE)

Run the executable in the root directory of this repository.<br>

# How to play Notepad-- (PYTHON)
Notepad-- can run on python. If you do not have python, install it (if you see the option `tcl/tk`, check that box).

Notepad-- also requires `tkinter`. Check if you have tkinter with `python3 -m tkinter`. If a small window appears, it is installed and you can skip the following steps. If a `ModuleNotFoundError` occurs, follow the instructions below to install tkinter.

## Installing Tkinter
### Windows

Re-run the python installer. Choose Modify and ensure the `tcl/tk and IDLE` is selected.

### macOS

The binary installers should include tkinter. If using Homebrew, run `brew install python-tk`

### Linux (Ubuntu / Debian):

Run `sudo apt-get install python3-tk`

### Linux (Fedora / RHEL):

Run `sudo dnf install python3-tkinter`


## Running Notepad-- with Python
Simply run `python3 run.py` to play notepad--.
